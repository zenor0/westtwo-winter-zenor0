# West2 final lab

## Requirements

+ node.js
+ FastAPI
+ MySQL

## Deploy

1. Configurate `api/config.py` file by your own setting. 
2. Install node requirements

  ```sh
  npm install
  ```
3. Launch vue
   run command `npm run dev`
4. Launch FastAPI
   run command `uvicorn api.main:app` to boot up back-end
